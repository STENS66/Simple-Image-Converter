## Description

**"Simple Image Converter"** is a secure, fast, and 100% offline desktop application that converts your images and documents efficiently. Officially validated and published on the **Microsoft Store**, it operates exclusively locally on the user's computer, ensuring maximum security and absolute data privacy.

### Features

- **User-friendly interface:** Designed to be easy to use, allowing anyone to convert images effortlessly.
- **Wide Format Support:** Supports multiple output formats, including **PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF**.
- **SVG Conversion:** Allows conversion of **SVG** files to **PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF**.
- **SVG Limitation:** The application does not support converting **PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF** files *to* **SVG** format.
- **XPS & OXPS Support:** Supports conversion of **XPS** and **OXPS** files to **PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF**. For **GIF** and **TIFF**, a single multi-page file is generated. For other image formats, one file is created per page of the original document.
- **PDF Compression Options:** When converting to PDF, you can choose from different compression levels (Max Quality, Light, Medium, Strong) to optimize your document size.
- **Image Quality Options (JPG, JPEG, WEBP):** When converting to these formats, you can select a quality level (Max Quality, High, Medium, Low) to adjust the compression rate and output file size.
- **No Administrator Rights Required:** Makes the application safer to use on any computer.
- **100% Offline:** **All operations are performed locally** without requiring an internet connection, ensuring user data privacy and security.
- **Fast & Efficient:** Avoids the need to upload and download images to external websites.
- **Batch Processing:** **Multiple files can be converted simultaneously**.
- **Folder Security:** The app warns you if the input and output folders are identical, allowing you to choose to continue or cancel to avoid overwriting original files.
- **Detailed Logging:** An `app.log` file is created and overwritten at each run, recording operations and any errors. This log can be viewed via a dedicated button in the GUI and is essential for diagnostics. For faster bug resolution, this file can be sent to support at app.sencie@gmail.com.

### Compatibility

Desktop application designed for Windows 10 and 11 (64-bit).

### Usage

To convert images with "**Simple Image Converter**":

1. Select a source folder containing the images and documents to convert. (Ensure only supported files are present in this folder for optimal conversion).
2. Select an output folder where the converted files will be saved.
3. Choose the desired output format from the available options.
4. If converting to **PDF**, **JPG**, **JPEG**, or **WEBP**, select the desired compression/quality level.
5. Click the "Convert Images" button to start the conversion.
6. The converted files will be saved in the specified output folder.
7. In case of issues, you can click "Show Log File" in the "Developer" section and send us this file to facilitate technical support.

### Advantages

- **Simplicity:** Designed to be easy to use, even for non-technical users.
- **Speed:** Local image conversion is faster as there is no need to upload or download files.
- **Total Control:** The user retains absolute control over their images and information with no risk of data leakage.
- **Improved Support:** Thanks to the detailed log file, diagnostics and problem resolution are more efficient.
