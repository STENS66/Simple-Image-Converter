## Description

"Simple Image Converter" is a simple and secure application that allows you to convert your image files and documents quickly and efficiently. It works only locally on the user's computer, ensuring maximum security and absolute privacy for your data.

## Preview

![Screenshot of the Simple-Image-Converter app](https://github.com/STENS66/Simple-Image-Converter/blob/main/images/Simple-Image-Converter.png?raw=true)

### Compatibility

Desktop application designed for:

* **Windows**: 10 and 11 (64-bit).
* **Linux**: Available via Snap package on **Ubuntu** and compatible distributions.

### Features

- The user interface is user-friendly and easy to use, allowing any user to convert images effortlessly.
- Supports multiple output formats, including PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF.
- Also allows the conversion of .SVG files to PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF formats.
- The application does not support the conversion of PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF files to .SVG format.
- Support for converting XPS and OXPS files to PDF, PNG, JPG, JPEG, WEBP, ICO, BMP, GIF, and TIFF formats. For GIF and TIFF formats, a single multi-page file is generated. For other image formats, one file is created per page of the original document.
- PDF compression options: When converting to PDF format, you can choose from different compression levels (Maximum Quality, Light, Medium, Strong) to optimize the size of your documents.
- Image quality options (JPG, JPEG, WEBP): When converting to these formats, you can select a quality level (Maximum Quality, High, Medium, Low) to adjust the compression ratio and output file size.
- No administrator rights required, making the application safer to use on any computer.
- All operations are performed locally, without requiring an internet connection, thus guaranteeing the privacy and security of user data.
- Conversion is fast and efficient, avoiding the need to download and upload images to external sites.
- Multiple files can be converted at the same time.
- Folder security: The application warns you if the input and output folders are identical, allowing you to choose to continue or cancel to avoid overwriting original files.
- Detailed logging: An app.log file is created and overwritten each time the application runs, recording operations and any errors. This log can be viewed at any time via a dedicated button in the graphical interface and is essential for diagnosis in case of problems. For faster bug fixes or troubleshooting, this file can be sent to support at app.sencie@gmail.com.

### Usage

To convert images with "Simple Image Converter":

1. Select a source folder containing the images and documents to be converted. (Ensure only supported files are present in this folder for optimal conversion).
2. Select an output folder where the converted files will be saved.
3. Choose the desired output format from the available options.
4. If you are converting to PDF, JPG, JPEG, or WEBP, select the desired compression level.
5. Click the "Convert Images" button to launch the conversion.
6. The converted files will be saved in the specified output folder.
7. In case of a problem, you can click "View Log File" in the "Developer" section and send us this file to facilitate technical support.

### Advantages

- **Security**: Works only locally and never communicates any data over the internet, guaranteeing information privacy.
- **Simplicity**: Designed to be easy to use, even for non-technical users.
- **Speed**: Local image conversion is faster because there is no need to download or upload files.
- **Total Control**: The user has full control over their images and information, without the risk of data leaks.
- **Enhanced Support**: Thanks to the detailed log file, diagnosis and problem resolution are more efficient.
